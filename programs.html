Training Program

Program Name: Machine Learning in Bioinformatics

Short Description of Program: This course will teach you the fundamentals of how AI is applied in the field of bioinformatics. You’ll explore how AI, machine learning, deep learning concepts are used in Protein/DNA function prediction and drug-target interaction, as well as for modeling complex biological systems.

You’ll learn the basics of collecting, analyzing, and modeling bioinformatics data using ML on this course. You’ll find out how to collect and explore bioinformatics data from public resources and then use AI to analyze and model this data to understand key biological processes better.

What topics will you cover?

  • Overview on bioinformatics
  • Machine Learning and how to apply it to bioinformatics
  • Feature engineering
  • Feature learning
  • Deep learning in bioinformatics
  • Analyzing and visualizing data
  • Research And Publication
